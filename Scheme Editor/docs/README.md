## Project Overview

**Scheme Card Editor** is a single-page web application that allows users to create custom trading-style cards. Users can:

1. Upload a custom image for the card artwork
2. Enter a card name (using Grandarena font)
3. Enter a card description (using Poppins font)
4. Download the finished card as JPG or PNG

### Key Principles
- **No login required** - Anonymous, instant usage
- **Client-side only** - No server processing needed
- **Simple & Modern** - Clean, professional interface
